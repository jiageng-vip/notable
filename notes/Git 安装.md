# Git 安装

```
【linux centos7】
1.查看系统是否已经安装git
	git --version
2.CentOS7 yum 安装git
	yum install git
3.安装成功
	git --version
4.卸载git
	yum remove git
```

```
【linux Debian】
apt install git-all
```

```
【macOS】
在 Mac 上安装 Git 有多种方式。 最简单的方法是安装 Xcode Command Line Tools。 Mavericks （10.9） 或更高版本的系统中，在 Terminal 里尝试首次运行 git 命令即可。
```
